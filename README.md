# laba_4
✔️ Кратко о работе программы

Программа позволяет работать со списком товаров. Она умеет: • добавлять обычные товары и товары со скидкой; • редактировать и удалять товары; • показывать товары в виде таблицы на экране; • экспортировать данные из БД в JSON-файл; • сортировать товары по конечной цене.
